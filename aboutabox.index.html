ABOUT A BOX

this is a box
you can put things in it
anything you want
a rock,
a photograph from years ago,
anything you can,
maybe nothing

the box will protect its contents
to two extents-
the box has a set capacity,
it can only hold so many things
and it is only so durable,
so look after the box
so it can look after you
because this is a special box;
i think the box loves you
